Format

copy: copy simple copies only URL links to the clipboard, copy text copies titles and URLs to the clipboard.

text: simple text format with page titles and URLs

csv: format that can be opened in MS Excel and other database applications

simple: just the URLs

gmail: the first gmail option is for prefilled compose window but limited to up to about 22 tabs due to gmail limitations, the second gmail option sets up a compose window and copies an unlimited amount of tab title and urls to the clipboard so that all you have to do is paste into the body of the gmail compose window. Output

These options are not used when the user chooses to copy to clipboard or gmail for output format.

screen: prints to the screen; use can use Cntrl-a Cntrl-c to copy the contents and Cntrl-v to paste them. On Macs, this is Cmmnd-a Cmmnd-c to copy and Cmmnd-v to paste.

file: prints to file which will be named based on the date with prefix "Tab_Snap" and extension ".csv" for csv formats or ".txt" for text format.
